# Judge

The critical observer for static sites. Monitor static files for changes, asynchronously inline critical css.

## Usage

Get Judged.

1.  `npm i -g judge`
1.  `judge html/glob/**/*.html path/to/styles.css &> judge.log &`
1.  Profit
